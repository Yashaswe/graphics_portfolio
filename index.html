Ray Tracer | Part 1

For this project, I built the core parts of a ray tracer in C++. I worked on adding ambient, diffuse, and specular lighting, handled multiple light sources, and rendered scenes with spheres. I created classes for materials, spheres, cameras, and scenes, and used C++ features like optional and vectors to manage ray intersections and lighting. The final output was a set of .ppm images showing different lighting effects, which will be the base for more complex features like reflections and texture mapping in future projects.

I implemented object rendering, color interpolation, and transformations using OpenGL functions. I added interactive controls for switching between wireframe and filled modes, enabling lights, and toggling shading types. Ray Tracer | Part 8

Built several advanced OpenGL projects. In Assignment 9 (Carousel), I created a detailed animated carousel scene using hierarchical modeling and 3D transformations. I implemented rotating, scaling, and translating parts like ducks, cats, horses, airplanes, and props, all interacting within a 1x1x1 bounding box. I also added camera controls, actor viewpoints, and lighting effects for rich visual interaction. I took this further by replacing OpenGL's internal transformation pipeline with my own matrix stack system. I implemented custom handling for identity, push/pop, translation, scaling, rotation, projection, and camera look-at functions, as well as matrix multiplication and vertex/normal transformation, effectively making OpenGL run entirely on my code. Together, these projects strengthened my understanding of 3D graphics, transformations, matrix operations, and building a full graphics engine from scratch.
