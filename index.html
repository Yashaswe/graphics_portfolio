Ray Tracer | Part 1

For this project, I built the core parts of a ray tracer in C++. I worked on adding ambient, diffuse, and specular lighting, handled multiple light sources, and rendered scenes with spheres. I created classes for materials, spheres, cameras, and scenes, and used C++ features like optional and vectors to manage ray intersections and lighting. The final output was a set of .ppm images showing different lighting effects, which will be the base for more complex features like reflections and texture mapping in future projects.
